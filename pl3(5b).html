Code to find sum of digits

Problem Statement: Write a code in any programming language that finds sum of ten numbers. 


	# Iterative Python program to find 
	# sum of digits of ten numbers
		
	def countDigit(n,arr): 
		count = 0
        while n > 0: 
            n--
            count+= arr[n]
		return count 
		
    # Driver Code    
    arr = [] 
    for i in range(10):
            arr[i]=int(input())
	print ("Sum of digits : % d"%(countDigit(10,arr)))

Sample Input

Enter an array of 10 integers:arr[]={10,10,10,10,1,1,1,1,10,1}

Sample Output

Sum of digits: 55

Variables used in the program

count, n,arr,i

Time Complexity

O(log10(n))
